# Automated Interview Question Generator

A web-based application that generates role-specific interview questions using a locally hosted AI model.

## Tech Stack
- Python (FastAPI)
- Ollama (Mistral LLM)
- HTML, CSS, JavaScript

## Features
- Generates MCQ and descriptive interview questions
- Uses a local, open-source AI model (no external APIs)
- Clean UI with loading states
- Runs completely offline after setup

## Notes
- The first AI response may take some time since the model runs locally.
